How to compile:
Just open the folder which tradutor.cpp is and use the following line on a terminal.
"g++ tradutor.cpp -o tradutor"
Of course, you can include optimizations.

How to run:
The .asm file, executable (tradutor) and functions.s (IA32 file) have to be at the same folder. Then, use:
"./tradutor file" (without the extension).
